для запуска нового проекта скопировать папки:

gulp
src
gulpfile.js
package.json
.gitignore
index.html 

открыть терминал в этой папке (shift+` или через меню правой кнопки)
набрать в консоли:  (npm i)  установятся все пакеты из файла package.json тех версий, которые указаны (latest - последняя версия)
 далее запускаем сборку в нужном режиме
npm run dev или npm run build
