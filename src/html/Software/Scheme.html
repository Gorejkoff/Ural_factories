Функционал программного обеспечения
Сервер DMR
Центральный элемент системы, который объединяет диспетчеров, радиостанции и шлюзы в единую инфраструктуру. Сервер выполняет маршрутизацию вызовов, передачу текстовых сообщений, управление соединениями и ведение журнала переговоров. Он также отвечает за интеграцию с внешними системами, включая навигационные платформы.
Диспетчер DMR
Программный модуль, предоставляющий операторам функционал управления радиосвязью. Позволяет диспетчерам осуществлять индивидуальные, групповые и общие голосовые вызовы, отправлять и принимать текстовые сообщения, контролировать регистрацию абонентов и управлять их доступом. Поддерживает интеграцию с картографическими сервисами для мониторинга местоположения абонентов.
Администратор DMR
Программный модуль, предоставляющий операторам функционал управления радиосвязью. Позволяет диспетчерам осуществлять индивидуальные, групповые и общие голосовые вызовы, отправлять и принимать текстовые сообщения, контролировать регистрацию абонентов и управлять их доступом. Поддерживает интеграцию с картографическими сервисами для мониторинга местоположения абонентов.
Шлюз
Шлюз выступает в качестве центральной точки входа для взаимодействия с оборудованием (ретрансляторы и стационарные радиостанции), обеспечивает единый интерфейс для доступа к различным его функциям и возможностям.
